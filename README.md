# 🐍 Python POO - Travaux Pratiques

This repository contains **Object-Oriented Programming (POO) exercises in Python** completed during the **1st year ISTA training**.

The goal of these practical works is to learn and apply the main concepts of **Object-Oriented Programming using Python**.

---

## 📚 Topics Covered

* Classes and Objects
* Attributes and Methods
* Constructors (`__init__`)
* Encapsulation
* Inheritance
* Multiple Inheritance
* Lists of Objects
* Object Management Systems

## 🚀 How to Run

1. Install **Python 3**
2. Clone the repository

```bash
git clone https://github.com/JABIR-YASSER/python-poo-tp.git
```

3. Navigate to a TP folder

```bash
cd TP6_HERITAGE
```

4. Run the Python file

```bash
python main.py
```
